## 📌 Overview

This is my **personal developer portfolio**, built using **Next.js** with a modern premium UI design.

The goal of this project is to present my work, experience, and skills in a **clean, interactive, and visually rich interface** using:

- Glassmorphism UI
- Floating sticker system
- Smooth animations
- Modern responsive layout

It is designed to feel like a **high-end developer showcase (Awwwards-style portfolio)**.

---

## 🚀 Features

- ⚡ Built with Next.js (App Router)
- 🎨 Premium UI with glassmorphism design
- 🧷 Floating sticker-based visual identity system
- 🎬 Smooth animations using Framer Motion
- 🖱️ Interactive hover & motion effects
- 📊 Animated tech stack section
- 📱 Fully responsive (mobile / tablet / desktop)
- 🌍 Deployed on Vercel
